# 01 — 需求规格说明书

## 项目概述

「视频会议助手」是一个面向办公场景的网页应用，解决会议信息分散、纪要整理耗时、待办遗漏等痛点。聚焦**会前准备**与**会后跟进**环节，提升视频会议效率。

## 目标用户

- 个人办公使用，无需多用户登录
- 日常需要参加多场视频会议的知识工作者

## 功能需求

### F1 — 会议列表管理

- 按时间线展示会议列表，按**今日 → 未来 → 历史（一周内）→ 已逾期**分段
- 今日会议突出高亮显示
- 每个会议卡片展示：时间、主题、负责人、参会人、链接、预计时长、状态
- 支持手动新建/编辑/删除会议
- 支持粘贴会议链接（腾讯会议/飞书/钉钉）智能解析标题和平台信息

### F2 — 会前准备

- **设备调试清单**：默认模板（摄像头、麦克风、扬声器、网络），逐项勾选
- **会议议程**：文本编辑，保存后实时更新
- **资料管理**：
  - 支持上传 Word（.doc/.docx）、PDF、PPT（.ppt/.pptx）、图片（.png/.jpg/.gif）
  - 文件列表展示（文件名、类型图标、上传时间）
  - 支持下载文件

### F3 — 会中助手

- **模拟转写**：文本输入框，输入文字后追加到转写区，带时间戳
- **重点标记**：点击"标记重点"按钮，将当前选中转写内容高亮标记
- **待办添加**：弹出表单，填写内容、负责人、截止日期，关联到当前会议

### F4 — 会后跟进

- **结束会议**：点击后会议状态变更为"已完成"
- **自动生成纪要**：聚合会议基本信息、转写重点、待办列表，生成纪要草稿
- **纪要编辑**：支持手动编辑纪要内容
- **待办管理**：会议关联的待办事项列表

### F5 — 待办总览

- 跨会议汇总所有待办事项
- 按状态筛选：全部 / 未开始 / 进行中 / 已完成 / 已逾期
- 支持修改状态、删除待办
- 一键导出 Excel（.xlsx）

## 非功能需求

- UI 风格：专业简洁，淡蓝色主调，左侧导航 + 右侧内容区
- 响应式：桌面端优先，兼顾平板
- 性能：页面加载 2 秒内
- 数据安全：本地单用户，数据库文件加密存储

## 不包含的功能（明确排除）

- 真实语音识别转写（第一版使用模拟方案）
- 多用户登录/注册
- 第三方平台 API 深度集成（OAuth）
- 邮件/消息通知
- 移动端适配
