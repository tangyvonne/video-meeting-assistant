# 04 — 数据库设计

## 概述

- **数据库**：SQLite 3
- **ORM**：Prisma 5.x
- **字符集**：UTF-8
- **文件位置**：`prisma/dev.db`

## ER 关系图

```
Meeting (1) ──< Document (N)
Meeting (1) ──< Transcription (N)
Meeting (1) ──< Todo (N)
Meeting (1) ──< Minutes (1)
```

## 表结构

### Meeting（会议）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | TEXT | PK, UUID | 主键 |
| title | TEXT | NOT NULL | 会议主题 |
| date | TEXT | NOT NULL | 会议日期 (YYYY-MM-DD) |
| startTime | TEXT | NOT NULL | 开始时间 (HH:mm) |
| endTime | TEXT | NOT NULL | 结束时间 (HH:mm) |
| duration | INTEGER | NOT NULL | 预计时长（分钟） |
| host | TEXT | NOT NULL | 负责人/主持人 |
| attendees | TEXT | NOT NULL | 参会人（逗号分隔） |
| link | TEXT | | 会议链接 |
| platform | TEXT | | 平台：tencent/feishu/dingtalk/other |
| status | TEXT | NOT NULL | upcoming/ongoing/completed/missed |
| agenda | TEXT | | 会议议程 |
| createdAt | DATETIME | NOT NULL | 创建时间 |
| updatedAt | DATETIME | NOT NULL | 更新时间 |

### Document（文件）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | TEXT | PK, UUID | 主键 |
| meetingId | TEXT | FK → Meeting | 所属会议 |
| fileName | TEXT | NOT NULL | 原始文件名 |
| filePath | TEXT | NOT NULL | 存储路径 |
| fileSize | INTEGER | NOT NULL | 文件大小（字节） |
| fileType | TEXT | NOT NULL | MIME 类型 |
| uploadedAt | DATETIME | NOT NULL | 上传时间 |

### Transcription（转写记录）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | TEXT | PK, UUID | 主键 |
| meetingId | TEXT | FK → Meeting | 所属会议 |
| content | TEXT | NOT NULL | 转写内容 |
| isHighlighted | BOOLEAN | DEFAULT false | 是否标记重点 |
| createdAt | DATETIME | NOT NULL | 创建时间 |

### Todo（待办事项）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | TEXT | PK, UUID | 主键 |
| meetingId | TEXT | FK → Meeting | 关联会议 |
| content | TEXT | NOT NULL | 待办内容 |
| assignee | TEXT | NOT NULL | 负责人 |
| dueDate | TEXT | NOT NULL | 截止日期 (YYYY-MM-DD) |
| status | TEXT | NOT NULL | pending/in_progress/done/overdue |
| createdAt | DATETIME | NOT NULL | 创建时间 |
| updatedAt | DATETIME | NOT NULL | 更新时间 |

### Minutes（会议纪要）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | TEXT | PK, UUID | 主键 |
| meetingId | TEXT | FK → Meeting, UNIQUE | 关联会议（一对一） |
| summary | TEXT | | 会议概要 |
| highlights | TEXT | | 重点内容汇总 |
| content | TEXT | | 完整纪要内容（可编辑） |
| createdAt | DATETIME | NOT NULL | 创建时间 |
| updatedAt | DATETIME | NOT NULL | 更新时间 |

## Prisma Schema

```prisma
// prisma/schema.prisma
generator client {
  provider = "prisma-client-js"
}

datasource db {
  provider = "sqlite"
  url      = "file:./dev.db"
}

model Meeting {
  id        String   @id @default(uuid())
  title     String
  date      String
  startTime String
  endTime   String
  duration  Int
  host      String
  attendees String
  link      String?
  platform  String   @default("other")
  status    String   @default("upcoming")
  agenda    String?
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt

  documents      Document[]
  transcriptions Transcription[]
  todos          Todo[]
  minutes        Minutes?
}

model Document {
  id         String   @id @default(uuid())
  meetingId  String
  fileName   String
  filePath   String
  fileSize   Int
  fileType   String
  uploadedAt DateTime @default(now())

  meeting Meeting @relation(fields: [meetingId], references: [id], onDelete: Cascade)
}

model Transcription {
  id            String   @id @default(uuid())
  meetingId     String
  content       String
  isHighlighted Boolean  @default(false)
  createdAt     DateTime @default(now())

  meeting Meeting @relation(fields: [meetingId], references: [id], onDelete: Cascade)
}

model Todo {
  id        String   @id @default(uuid())
  meetingId String
  content   String
  assignee  String
  dueDate   String
  status    String   @default("pending")
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt

  meeting Meeting @relation(fields: [meetingId], references: [id], onDelete: Cascade)
}

model Minutes {
  id         String   @id @default(uuid())
  meetingId  String   @unique
  summary    String?
  highlights String?
  content    String?
  createdAt  DateTime @default(now())
  updatedAt  DateTime @updatedAt

  meeting Meeting @relation(fields: [meetingId], references: [id], onDelete: Cascade)
}
```
