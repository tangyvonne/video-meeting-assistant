# 03 — UI 设计规范

## 色彩系统

### 主色调：淡蓝色系

| 用途 | 色值 | 说明 |
|------|------|------|
| Primary | `#2563EB` | 主按钮、链接、激活态 |
| Primary Light | `#DBEAFE` | 浅蓝背景、hover态 |
| Primary Dark | `#1D4ED8` | 按钮按压态 |
| Background | `#F8FAFC` | 页面主背景 |
| Card Background | `#FFFFFF` | 卡片白色背景 |
| Text Primary | `#1E293B` | 正文文字 |
| Text Secondary | `#64748B` | 辅助文字 |
| Border | `#E2E8F0` | 边框、分割线 |
| Success | `#16A34A` | 已完成状态 |
| Warning | `#F59E0B` | 进行中/逾期状态 |
| Danger | `#DC2626` | 删除按钮、已逾期标签 |
| Muted | `#94A3B8` | 未开始状态 |

### Tailwind 主题配置

```ts
// tailwind.config.ts 中扩展
colors: {
  primary: {
    50:  '#EFF6FF',
    100: '#DBEAFE',
    200: '#BFDBFE',
    500: '#3B82F6',
    600: '#2563EB',
    700: '#1D4ED8',
  }
}
```

## 字体

- **字体栈**：`"PingFang SC", "Microsoft YaHei", "Helvetica Neue", Arial, sans-serif`
- **标题**：`font-semibold`，大小 24px / 20px / 18px
- **正文**：`font-normal`，大小 14px / 16px
- **辅助文字**：`text-sm`（14px），`text-secondary`

## 布局

### 整体布局

```
┌──────┬──────────────────────────────────────┐
│ 导航栏 │  顶部栏                               │
│ 200px  ├──────────────────────────────────────┤
│      │                                       │
│ 📅   │  内容区域                              │
│ 会议  │  (最大宽度 1200px，居中)                │
│      │                                       │
│ 📝   │                                       │
│ 待办  │                                       │
│      │                                       │
└──────┴──────────────────────────────────────┘
```

- 左侧导航栏：固定 200px 宽度
- 右侧内容区：自适应，最大宽度 1200px，居中
- 导航项包含图标+文字，当前页高亮

### 间距

- 页面 padding：`p-6`（24px）
- 卡片间距：`gap-4`（16px）
- 组件内边距：`p-4`（16px）

## 组件规范

### 按钮

- 主按钮：`bg-primary-600 text-white rounded-lg px-4 py-2`
- 次按钮：`border border-gray-300 rounded-lg px-4 py-2`
- 危险按钮：`bg-red-600 text-white rounded-lg px-4 py-2`
- 小按钮：`text-sm px-3 py-1`

### 卡片

- `bg-white rounded-lg border border-gray-200 shadow-sm p-4`

### 表单

- 输入框：`border border-gray-300 rounded-md px-3 py-2 w-full`
- 标签：`text-sm font-medium text-gray-700 mb-1`

### 标签/徽章

- 会议状态标签：小圆角徽章，颜色区分状态
  - 进行中：绿色
  - 即将开始：蓝色
  - 已结束：灰色
  - 已逾期：红色

### 弹窗 (Dialog)

- 使用 shadcn/ui Dialog 组件
- 遮罩层：`bg-black/50`
- 弹窗内容：`bg-white rounded-lg shadow-xl max-w-md`

## 响应式断点

| 断点 | 宽度 | 适配 |
|------|------|------|
| Desktop | ≥1024px | 完整布局 |
| Tablet | 768-1023px | 导航栏收窄为图标 |
| Mobile | <768px | 暂不优化（第一版） |

## 图标

- 使用 Lucide Icons（shadcn/ui 自带图标库）
- 导航图标：20px
- 操作图标：16px
